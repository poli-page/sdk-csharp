This page tracks breaking changes between major versions of the `PoliPage` NuGet package. For the full per-release diff, see the [changelog](../changelog/).

## v1.0

The first stable release. The full v1.0 surface and the renames from the pre-1.0 line are documented in [`MIGRATION.md` in the repo](https://github.com/poli-page/sdk-csharp/blob/main/MIGRATION.md).

Short version: every render call now goes through a namespace (`Render.PdfAsync`, `Render.PreviewAsync`, `Render.DocumentAsync`), all PDF-producing methods require `ProjectModeInput`, and inline mode is supported only on `Render.PreviewAsync`.

## Example

```csharp
// pre-1.0
var pdf = await client.RenderAsync(input);

// 1.0+
byte[] pdf = await client.Render.PdfAsync(input);
```
